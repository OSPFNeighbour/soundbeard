Soundbeard

A simple REST based soundboard in nodeJS.

List all available snippets

GET /list

Sample response:
{ "foo.mp3": "http://127.0.0.1:8080/play/foo.mp3", "bar.wav": "http://127.0.0.1:8080/play/bar.wav" }

Play a single snippet

GET /play/:snippet (:snippet equals the filename i.e. foo.mp3)

Sample response:
{"playing":"foo.mp3"}

Stop playback

GET /stop
